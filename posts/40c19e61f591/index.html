这是一块DFROBOT产的5’’ 800x480 TFT 电容屏

默认是按长边方向显示的，如果竖屏使用的话，就需要调整屏幕方向和触摸方向

二、屏幕旋转

三、触摸旋转

如果只是屏幕进行了旋转的话，触摸还是保持默认方向，就无法正常操作了，需要调整与屏幕方向一致才行

编辑配置

在libinput touchscreen catchall部分添加一行Option "CalibrationMatrix" "0 1 0 -1 0 1 0 0 1"

重启即可
