树莓派pi和root账户密码

发表于 2018-03-11 更新于 2024-09-05 本文字数： 380 阅读时长 ≈ 1 分钟

pi账户

pi是树莓派的默认账户，默认密码为 raspberry

修改默认密码

sudo raspi-config

选择的一项 Change User Password，输入两次新密码即可完成密码修改

root账户

树莓派的root账户在默认情况下是禁用的，且密码为空，所以root账户在未启用时，无论输入什么密码均是错误密码

设置root密码

sudo passwd root

输入两次新设置的密码后提示密码更新成功

pi和root账户切换

输入

su root

和密码后便可登录root账户

输入

su pi

后便可切换到pi账户

如何锁定root账户

输入

sudo passwd --lock root

锁定root账户后，我们再尝试用root账户登录，会提示Authentication failure(验证失败)

当我们输入

sudo passwd --unlock root

后，便可解锁root账户

再次登录root账户成功
